## ReportAssistant

NDC Template Tools 國發會範本工具

## Linux產生ReportAssistant.oxt
	1. 在clone or download 選擇 Download ZIP
	2. 解壓縮檔zip
	3. 進入解壓目錄
	4. 執行make

## Windows產生ReportAssistant.oxt
	1. 在clone or download 選擇 Download ZIP
	2. 選擇檔案按右鍵~選Extract All~
	3. 進入解壓目錄
	4. 選擇runoxt.bat~doubleclick 或按 Enter 執行

##　Using python to create ext

```
python create_ext.py
```
